TG-WebApp-Proxy lets you use Telegram right from your web browser. It connects directly to Telegram servers without needing extra software. This makes chatting, downloading, and managing files easy and fast.

---

## 🔎 What is TG-WebApp-Proxy?

TG-WebApp-Proxy is a full-featured Telegram client that runs inside your browser. It uses a secure method to connect called MTProto WebSocket. You do not need to install anything on your computer except your web browser.

This app also supports a Bot Mode. You can link a Telegram bot via its token and download files without limits on size. The app splits downloads into parts to make them faster.

## ⚙️ Basic Setup and Usage

After starting the app, you will see options similar to the Telegram mobile or desktop apps.

- Enter your phone number and confirm via the code Telegram sends you.
- You can join chats, view your contacts, and browse channels.
- Use the **Bot Mode** by entering your bot token to download large files directly.
- Paste any Telegram `t.me` link to find files and save them on your computer.
- Your messages and files will stay available even if you close the browser, thanks to local storage.

## 🖥️ How to Use Bot Mode

1. Create a bot on Telegram by talking to [@BotFather](https://telegram.me/BotFather).
2. Copy the bot token provided by BotFather.
3. Open TG-WebApp-Proxy.
4. Paste the bot token in the Bot Mode section.
5. Use the bot to download files directly from messages.
6. The app lets you reply and manage bot messages from the browser.
